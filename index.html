Django Abstract Startpoint

Mostly whenever I start a new django project, i use django's Abstract base user rather than the default user. And instead of writing the codes that enables the AbstracBaseUser all the time, I created Django Abstract Startpoint. With Django Abstract Starpoint, I just have to clone it to my project and run the "AUTO_START" command and it would write all that is necessary for me to use Abstract Base User in my project while i just go straight to writing the important stuff. I also made it an open source program so other developers can use it too.
